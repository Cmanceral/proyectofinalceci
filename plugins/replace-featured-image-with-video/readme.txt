== Description ==
Replace Featured Image with Video is a simple plugin to replace the featured image in your wordpress post or page with a video. Supports Youtube, Dailymotion, Vimeo and many other sites that provide embedd code.. just don't forget to edit the embedd code to the size of your featured image 

This plugin will attempt to override any existing featured image so removal or deletion should not be necessary.

Tested up to 3.4.1


#### Functionality
* Replace the featured image in your wordpress website with a video.


== Installation ==
Follow the steps below to install the plugin.

1. Upload the files to your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There will now be a box above the featured image option in both posts and pages - edit the embedd code height and width to the size of your featured image.
